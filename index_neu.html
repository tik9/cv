Remove a password from an Excel VBA Code as seen here
N.B. You do not need to know the password. You must have write access to the xls.
  1. Open the xls file with Notepad
  2. Search for 'DPB=' and replace it with 'DPx='
  3. Save the file, Open the file, open the VB Editor, open modules will not work (error 40230) properly
  4. Save the file as xlsm
